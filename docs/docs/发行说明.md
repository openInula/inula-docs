## 新特性

**openInula**

* 支持响应式 API（9月底发布β版本）
* 支持状态管理器 inulaX
* 兼容 React API，支持类式组件和函数式组件

**create-inula**

- 支持选择 Simple-app 模板
- 项目模板支持 webpack 和 vite 两种打包方式
